# JavaScript Array（数组）

- **数组对象的作用是：**使用单独的变量名来存储一系列的值。

## 定义数组

数组对象用来在单独的变量名中存储一系列的值。
使用关键词 new 来创建数组对象。		
例：	
	
```javascript
var myArray=new Array()
```

## 赋值	
	
例：
		
```javascript
var mycars=new Array()
mycars[0]="Saab"
```

或	
	
```javascript
var mycars=new Array("Saab","Volvo","BMW")
```
